# Retro Walk: Historical Map Stroller 🗾👟

**Retro Walk** is a unique location-based application built with **Flutter** that allows users to stroll through Japanese streets while viewing historical maps. Experience a "time-traveling" walk by comparing present-day scenery with maps and layouts from different eras.

## 🌟 Overview
This service is available exclusively in **Japan**, utilizing specialized historical map data covering various regions across the country.

- **Time-Travel Maps:** Seamlessly switch between maps from the **1950s, 1960s, 1970s, 1980s**, and the present day.
- **GPS Integration:** Real-time display of your current location on historical maps, allowing you to identify what stood in your spot decades ago.
- **Historical Exploration:** Discover old building footprints, vanished landmarks, and the evolution of urban planning.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Complex integration of GIS (Geographic Information System) data and GPS synchronization to align historical map tiles with modern coordinates.
- **Target Market:** Japan 🇯🇵

*Note: This repository contains the core logic and source code. Assets (map tiles, historical data, and localization files) are excluded to protect intellectual property.*
